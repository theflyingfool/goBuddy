# Feature history

Snapshot of [`current.md`](../current.md) taken at each release cut, so past
feature specs stay readable after `current.md` moves on. Convention: at each
`npm run version:bump`, copy `docs/features/current.md` to
`docs/features/history/vX.Y.Z.md` *before* updating `current.md` for the new
release.

Empty today — no prior release has a feature spec that differs from the
current one yet. The first snapshot will be taken at the *next* version bump
after this one.
